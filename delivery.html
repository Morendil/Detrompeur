Le Détrompeur

Le Détrompeur est un outil qui vous facilite la lecture d'articles utilisant le jargon Agile. Vous lisez par exemple: "plusieurs itérations peuvent être lancées simultanément ou les unes à la suite des autres".

(Il se trouve que c'est un contre-sens, mais si vous avez un doute là-dessus, ce n'est pas toujours évident de savoir où chercher une information fiable.)

Il suffit de cliquer sur le lien "détrompeur" ci-dessus pour que ces termes soient surlignés dans la page Web que vous consultez, et lorsque vous déplacez votre souris sur ces termes vous pourrez lire un extrait de la description qu'en fait le Référentiel de l'Institut Agile. En cliquant, vous pouvez lire la fiche complète. (Elle vous apporte plus de précisions, par exemple les erreurs courantes dans l'interprétation ou l'application de chaque pratique.)

Oui, mais... en dehors de celle-ci, les pages que vous lisez n'affichent pas ce lien?

Qu'à cela ne tienne: ce lien est un "bookmarklet", il peut fonctionner même quand vous aurez quitté cette page. Il vous suffit de faire glisser ce lien vers la barre d'adresse de votre navigateur, vous pourrez ensuite le cliquer sur n'importe quelle page. (Voir ci-dessous une vidéo illustrant un exemple d'utilisation d'un "bookmarklet".)

Essayez maintenant: déposez le Détrompeur dans votre barre d'adresses, puis suivez ce lien vers un article parlant de pratiques Agiles. Dès que la page est affichée, cliquez sur le Détrompeur. Bonne lecture!
